## 📂 需要的文件

下载 `tianai-captcha-web-sdk-1.5.3.zip` 后，解压并将以下文件放到对应目录：

```
plus-ui-ts/public/tac/
├── js/
│   └── tac.min.js          ← 从 dist/js/tac.min.js 复制
├── css/
│   └── tac.css             ← 从 dist/css/tac.css 复制
└── 下载说明.md (本文件)
```

## 🔧 安装步骤

### 1. 下载SDK

访问上述地址，下载 `tianai-captcha-web-sdk-1.5.3.zip`

### 2. 解压文件

解压后目录结构：
```
tianai-captcha-web-sdk-1.5.3/
├── dist/
│   ├── js/
│   │   └── tac.min.js
│   ├── css/
│   │   └── tac.css
│   └── ...
└── ...
```

### 3. 复制文件

将以下文件复制到项目中：

```bash
# 复制JS文件
dist/js/tac.min.js  →  plus-ui-ts/public/tac/js/tac.min.js

# 复制CSS文件
dist/css/tac.css    →  plus-ui-ts/public/tac/css/tac.css
```

### 4. 验证安装

启动前端项目后，访问：
- http://localhost/tac/js/tac.min.js
- http://localhost/tac/css/tac.css

如果能正常访问，说明安装成功！

## 🐛 常见问题

### 1. 404错误：找不到tac.min.js

**原因**: 文件未正确放置

**解决**: 
1. 检查文件路径是否正确
2. 确保文件名完全一致（区分大小写）
3. 重启前端开发服务器

### 2. 验证码不显示

**原因**: CSS文件未加载

**解决**:
1. 检查 `tac.css` 是否存在
2. 打开浏览器控制台查看是否有CSS加载错误
3. 清除浏览器缓存

### 3. 控制台报错：_TACBuilder is undefined

**原因**: JS文件未正确加载

**解决**:
1. 检查 `tac.min.js` 是否存在
2. 检查文件大小是否正常（应该有几百KB）
3. 重新下载SDK文件
