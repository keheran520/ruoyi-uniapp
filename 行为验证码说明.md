# 行为验证码使用说明

## 概述

本项目支持在移动端使用行为验证码，采用了与B端相同的 TAC（Tianai-Captcha）验证码方案。

## 平台支持

### H5 平台
- 使用 TAC 官方 SDK
- 浏览器中直接显示验证码弹窗
- 滑动拼图验证

### 小程序/APP 平台
- 使用自定义 `BehaviorCaptcha.vue` 组件
- uview-plus UI 风格
- 触摸滑动验证

## 文件结构

```
ruo-yi-uni-app-plus/
├── public/
│   ├── load.min.js                          # TAC 加载器（H5）
│   └── tac/                                  # TAC 资源文件
│       ├── css/tac.css                      # TAC 样式
│       ├── js/tac.min.js                    # TAC 核心脚本
│       └── images/                          # TAC 图片资源
├── src/
│   ├── components/
│   │   └── BehaviorCaptcha/
│   │       └── BehaviorCaptcha.vue          # 自定义验证码组件（非H5）
│   ├── utils/
│   │   └── behaviorCaptcha.ts               # 验证码工具类
│   └── pages/
│       └── login.vue                        # 登录页面
```

## 使用方式

### H5 平台

在 H5 平台，会自动使用 TAC 验证码：

```javascript
import { showBehaviorCaptchaModal } from '@/utils/behaviorCaptcha'

// 显示验证码并获取 captchaId
try {
  const captchaId = await showBehaviorCaptchaModal({
    tenantId: '000000'
  });
  
  // 验证成功，使用 captchaId 发送请求
  console.log('获取到验证码ID:', captchaId);
} catch (error) {
  // 用户取消或验证失败
  console.log('验证失败');
}
```

### 小程序/APP 平台

在非 H5 平台，使用自定义组件：

```vue
<template>
  <view>
    <!-- 行为验证码组件 -->
    <BehaviorCaptcha 
      v-model="showCaptcha" 
      :tenantId="tenantId"
      @success="handleCaptchaSuccess"
      @fail="handleCaptchaFail"
      @close="handleCaptchaClose"
    />
  </view>
</template>

<script setup>
import { ref } from 'vue';

const showCaptcha = ref(false);
const tenantId = ref('000000');

function handleCaptchaSuccess(captchaId) {
  console.log('验证成功，captchaId:', captchaId);
  // 使用 captchaId 发送请求
}

function handleCaptchaFail() {
  console.log('验证失败');
}

function handleCaptchaClose() {
  console.log('用户关闭');
}
</script>
```

## 登录页面实现

登录页面已经集成了三种登录方式，并自动根据平台选择合适的验证码方案：

### 密码登录
- 用户名 + 密码
- 图形验证码（如果启用）

### 邮箱登录
- 邮箱地址
- 邮箱验证码（需要先完成行为验证）

### 号码登录
- 手机号
- 短信验证码（需要先完成行为验证）

## 条件编译

项目使用 uni-app 条件编译来区分不同平台：

```javascript
// #ifdef H5
// H5 平台代码
const captchaId = await showBehaviorCaptchaModal({ tenantId });
// #endif

// #ifndef H5
// 非 H5 平台代码
showCaptcha.value = true;
// #endif
```

## API 接口

### 生成验证码
- **接口**: `/dev-api/captcha/gen`
- **方法**: POST
- **参数**: `{ type: 'SLIDER' }`

### 校验验证码
- **接口**: `/dev-api/captcha/check`
- **方法**: POST
- **参数**: `{ id: string, data: string }`

### 发送邮箱验证码
- **接口**: `/resource/email/code`
- **方法**: GET
- **参数**: `{ email, businessType, captchaId, tenantId }`

### 发送手机验证码
- **接口**: `/resource/phoneverify/code`
- **方法**: GET
- **参数**: `{ phonenumber, businessType, captchaId, tenantId }`

## 注意事项

1. **H5 平台**: TAC 验证码需要在浏览器环境中运行，依赖 DOM API
2. **资源文件**: 确保 `public/tac` 目录下的资源文件完整
3. **跨域配置**: 如果 API 地址不同，需要配置代理或 CORS
4. **租户ID**: 根据实际业务传递正确的租户ID
